# Alexa Skill with Gemini AI

ノーコードでアレクサにGeminiのスキルを作成することができます。

## セットアップ手順

1. `lambda/keys.template.js`を`lambda/keys.js`にコピー
2. `keys.js`内のAPI keyを自分のものに置き換え
3. その他の設定を必要に応じて調整

## 注意
`keys.js`は`.gitignore`に含まれており、APIキーなどの機密情報はGitHubにアップロードされません。
